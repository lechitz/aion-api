# Personas (aion-api)

## Purpose

This folder lists the local personas used to specialize implementation, review, debugging, and documentation work inside `aion-api`.

## What You Will Find Here

- `aion.docs.md`
- `aion.architect.md`
- `aion.platform.md`
- `aion.tester.md`
- other task-specific personas used by the local control center

## How To Use

1. identify the task type
2. choose the persona that best matches the work
3. follow its constraints before editing code or docs
